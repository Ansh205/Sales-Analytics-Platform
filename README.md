# Sales Analytics Platform

## Project Overview

This project is an end-to-end Sales Analytics Platform built using:

- Python
- PostgreSQL
- SQL
- Power BI
- FastAPI
- Docker

## Features

- Executive Dashboard
- Customer Dashboard
- Product Dashboard
- Profit Analysis
- Regional Analysis
- Forecasting
- Drill-through Reports
- Power BI Embedded
- REST APIs
- Cloud Deployment
